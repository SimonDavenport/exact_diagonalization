Detailed Description

template<typename T, class B>
class diagonalization::HamiltonianBase< T, B >

The Hamiltonian class contains common utility functions for matrix diagonalization.

The Hamiltonian is constructed from the outer product:

|leftVector><rightVector|

Where <rightVector| is given by applying the Hamiltonian to the full Fock space and |leftVector> is the full set of Fock states

The template parameter T denotes the data type of the matrix elements The template parameter B denotes the data type for the Fock basis
