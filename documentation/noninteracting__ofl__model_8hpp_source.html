64  //! \brief The NonInteractingOflModel class contains a matrix representation of
65  //! the single particle optical flux lattice Hamiltonian as a 2D difference equation
66  //! for Block-type eigenfunctions. The class contains funcitons to populate and
67  //! diagonalize that matrix
68  //!
69  //! This class sets up the difference equation problem defined by taking the single
70  //! particle Hamiltonian (eq. (1) in PRL 109,265301). First we perform a unitary
71  //! transformation with U = {{exp(-ik_3.r),0},{0,exp(ik_3.r)}}. This has the
72  //! effect of making the Bx+iBy part (off diagonals) expressible in terms of
73  //! the reciprocal lattice vectors, at the expense of shifting the kinetic term.
74  //!
75  //! Then we provide the following Bloch wave function ansatz
76  //!
77  //! \[ |u^{n,k}_{\sigma} > = sum_{G} c^{n,k}_{G,sigma} exp(i(k+G).r) |sigma> \]
78  //!
79  //! Where n is the band index, k is a position in 2D k-space, {G} is the set of reciprocal
80  //! lattice vectors, sigma is a spin index (up or down here) and r is the position in real
81  //! space. Inserting this into Eq (1) leads to a matrix relation between the
82  //! coefficients c^{n,k}_{G,sigma}. This is an infinite matrix in general,
83  //! however, considered in a reduced zone scheme it becomes evident that if we only
84  //! wish to determine the lowest part of the band structure then it's sufficient
85  //! to solve a truncated equation with x and y cut-offs in the number of reciprocal lattice
86  //! periods.
87  //!
88  //! The resulting coefficients are calculated for a given set of Hamiltonian
89  //! parameters, and at a given position in k-space. This result can be used
90  //! to build up a map of the energy bands throughout k-space, as well as the
91  //! values of coefficents that can later be used to construct the interacting
92  //! Hamiltonian.
93  //////////////////////////////////////////////////////////////////////////////////
