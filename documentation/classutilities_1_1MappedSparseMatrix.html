Detailed Description

template<typename T>
class utilities::MappedSparseMatrix< T >

A class template to implement the mapped sparse matrix format.

This class uses the std::map container to represent a sparse matrix: The map associated a combined row/column index index i*nbrCols+j with a matrix element.

Template parameter T is the matrix element type e.g. double of dcmplx

NOTE: the default hash function for google::denst_hash_map or sparse_hash_map is std::tr1::hash<unsigned long>. The Murmur Hasher function is far superior

Class template pre-declaration - allows CrsSparseMatrixType constructor to be called from a MappedSparseMatrix
