Detailed Description

template<typename T>
class diagonalization::SpinlessFermionHamiltonian< T >

The SpinlessFermionHamiltonian class generates and contains a set of basis states that are a bitwise representation of fermions occupying a set of orbitals.

Fock states are represented in ascending order c^+_0 c^+_2 c^+_5 etc. and are encoded in uint64_t binaries

The template parameter T denotes the data type of the matrix elements
