Set the type of index to address the sparse matrix with (e.g. uint_64_t)

Funciton to generate the set of two-body Haldane Pseudopotentials for the Coulomb interaciton in the Sphere goemetry. The analytic form for the lowest Landau level is.

V_L = 2/R [4Q-2L]C[2Q-L] * [4Q+2L+2]C[2Q+L+1] / ([4Q+2]C[2Q+1])^2

Where R is the sphere raduis, given by sqrt(Q)
