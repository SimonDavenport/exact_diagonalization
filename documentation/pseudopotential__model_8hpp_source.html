6 //! This file defines a class to store a single Landau level FQHE
7 //! Haldane pseduopotential model for in the sphere geometry.
8 //!
9 //! The class allows for either dense or sparse matrix storage, and within
10 //! that it allows for a mapped sparse matrix or compressed column format.
11 //! Any format can be selected to populate the matrix (though the mapped
12 //! sparse matrix format is typically faster - and the dense format will
13 //! use a lot more memory). The storage format will automatically
14 //! be converted to CCS when any sparse diagonalization routine is called.
